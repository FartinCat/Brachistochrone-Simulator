The Brachistochrone Problem (Greek: brachistos = shortest, chronos = time), first posed by Johann Bernoulli in 1696, asks: "Given two points A and B in a vertical plane, what is the curve of fastest descent for a frictionless particle sliding under gravity alone?"

The Cycloid — Brachistochrone Solution

The answer is the cycloid — the curve traced by a point on the rim of a circle of radius R rolling along a horizontal line. It is defined parametrically by:

x(θ) = R (θ − sin θ) y(θ) = R (1 − cos θ) θ ∈ [0, π] where R = H/2, H = vertical drop

Transit Time Formulae

Cycloid (analytical closed form):

T_cycloid = π √(R/g) where R = H/2

Straight Line (analytical):

L = √(X² + H²), X = πH/2 (cycloid horizontal reach) T_straight = √( 2L² / (g·H) )

Circular Arc (numerical integration):

Radius Rc = (X² + H²) / (2H) T_arc = ∫ ds / √(2g·y(s)) — evaluated over 600 arc segments

Derivation via Calculus of Variations

Using energy conservation, speed at depth y is v = √(2gy). The transit time functional is:

T = ∫ √(1 + y'²) / √(2gy) dx

Since the integrand F(y, y') is independent of x, the Euler-Lagrange equation reduces to the Beltrami identity: F − y'(∂F/∂y') = C, which yields:

y(1 + y'²) = k (constant) — the differential equation of the cycloid.

Tautochrone Property

The cycloid is also a tautochrone (Huygens, 1659): a ball released from any point on a cycloidal bowl reaches the bottom in the same time T = π√(R/g), independent of the release position. This led Huygens to design the first isochronous pendulum clock.

Expected Ranking

T_cycloid < T_arc < T_straight (always, for any H and g)
